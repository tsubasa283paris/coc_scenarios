# CoCシナリオ：玉虫色の絆

推奨技能：目星, 図書館  
人数：3人～(戦闘がキツイので)  
時間：3時間

ガッツリ奉仕種族と戦闘して、ガッツリ魔術に触れるシナリオ。  
ヨグ・ソトースがバック。

## 背景

今からおよそ15年前のこと。ごく普通の子持ち家庭の乙坂家は、車で買い物に出かけていた。  
林道を走っている最中、道を遮る倒木に行く手を阻まれる。  
どかそうと車を降りたところで何者かに取り押さえられ、瞬きをした瞬間には親子3人とも見知らぬ場所にいた。  
魔術師・倉本信二の生贄の館であった。  
末っ子の乙坂沙奈は、友達と遊びに出かけていた。

## 戦闘ルール

DEX順の行動。  
PC同士の声掛けはシームレスに、ターン消費せずに可能です。  
各PCは1ターンにマンハッタン距離で3マスまで移動可能です。直進の場合は5マスまで。  
移動と同時に以下のいずれかの行動が可能です。  

- 攻撃
- アイテムを渡す(3マス以上離れている場合は投擲で成功判定;失敗した場合ランダムな位置にアイテムを設置してください)
- <<目星>> <<聞き耳>>ほか適当な技能

敵キャラクターは近接攻撃しかない場合は1ターンに`DEX / 4` (小数点以下切り上げ) マス移動します。

## 魔術師の館・屋外  

```

```

正面入り口の扉からのみ入ることができます。  
窓はすべてカーテンで閉ざされており、中の様子は伺うことができません。  

- 扉を調べる  
  観音開きの立派な正面扉です。  
  扉には鍵はかかっておらず、引けば開きます。  
  <<聞き耳>>を立てても何も聞こえません。  

- 窓を割ろうとする / 窓をよく調べる  
  窓の表面にシャボン玉のような虹色の文様が浮かんでいることに気が付きます。  
  その上で窓を割ろうとしても、びくともしません。
